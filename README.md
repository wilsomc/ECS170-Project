# Getting Started
A step-by-step guide on running this project.

> MacOS
1. Create a new virtual environment
```
python3 -m venv .venv
```
2. Installing required dependencies
```
pip3 install -r requirements.txt
```
3. On PyCharm simply double-click the file you want to compile, change run configurations to "Selected File", and press the run button!


4. When finished run this command to exit (.venv).
```
deactivate
```

Opening the virtual environment.
```
source .venv/bin/activate
```
